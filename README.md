# Aid

## Environment

```bash
# Create environment
conda env create -f environment.yml
# Activate environment
conda activate aid
# Deactivate environment
conda deactivate
```

## Mirror

```bash
pip config set global.index-url https://mirrors.cloud.tencent.com/pypi/simple
```

## Development

```bash
make install
make dev
```

## Start

```bash
make start
```
